# Chapter XII: Adversaries

Quick reference for all adversary stat blocks from the Edge of the Empire Core Rulebook, organised by threat level.

**Key:** BR = Brawn, AG = Agility, INT = Intellect, CUN = Cunning, WIL = Willpower, PR = Presence

---

## Minions

Minions do not suffer strain (wounds instead), do not possess skill ranks individually (gain ranks when acting as a group), fight as a group sharing a combined wound threshold, and are killed by Critical Injuries.
